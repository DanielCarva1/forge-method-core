# Forge Method 1.25.0

Forge Method now updates itself before normal startup when installed from the Git marketplace.

## New

- Self-update runs from the launcher before `start`, `preflight`, `guide`, and `resume`.
- Update failures do not block startup; Forge continues with the local runtime.
- Patch notes appear once per installed version and stay out of stdout for JSON commands.
- Legacy skill-only installs show a one-time hint to migrate to the Git marketplace install.

## Notes

Forge continues the current start after updating. If a future update changes the skill text or plugin manifest, the runtime still continues in the same chat and prints a short note that a later new thread can load refreshed skill instructions fully.
