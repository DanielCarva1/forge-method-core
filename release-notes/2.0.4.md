# Forge Method Core 2.0.4

**Released:** 2026-06-23

Hotfix for Bash verification argument parsing.

## Fix

- `scripts/verify-fast.sh` now validates values for `--test`, `--match`, `--report`, `--junit`, `--workers`, and `--timeout` before reading `$2` under `set -u`.
- `scripts/verify-all.sh` now validates values for `--report`, `--junit`, `--workers`, and `--timeout` before reading `$2` under `set -u`.
- Missing option values now return exit code `2` with `Missing value for --flag` instead of a Bash `unbound variable` crash.
- Added regression coverage for missing values in both shell scripts.

## Upgrade

```txt
$forge-update
```
