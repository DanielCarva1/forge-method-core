# Forge Method Core 2.0.1

**Released:** 2026-06-23

Hotfix for v2.0.0.

## Fix

- `agent_id` not threaded through `append_request` in fleet-mode `handoff` and `checkpoint`. Worker requests showed `"default"` instead of the actual agent identity. Both commands now pass `agent_id=_resolve_agent_id(args)` to `append_request`.

$forge-update
