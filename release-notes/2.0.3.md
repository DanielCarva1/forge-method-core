# Forge Method Core 2.0.3

**Released:** 2026-06-23

Hotfix for Bash verification.

## Fix

- `scripts/verify-fast.sh` now works under Bash `set -u` / nounset when no `--test` or `--match` filters are passed.
- Empty `tests` and `matches` arrays are checked before expansion, avoiding `unbound variable` failures in Git Bash and strict Bash environments.
- Added a regression test that executes `verify-fast.sh` with no filters through Bash and a fake Python runner.

## Upgrade

```txt
$forge-update
```
