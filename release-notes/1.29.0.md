# Forge Method 1.29.0

Guided workflow depth for richer human routing and compact agent handoff.

## Highlights

- Added broader Guidance Engine and Help Oracle behavior so substantive human intent routes through explicit workflows before stale state can dominate.
- Added Builder Factory workflows for module ideation, agent building, workflow building, module packaging, and module validation.
- Added Project Configuration, Override Model, generated Capability Index, Persona Lens overlays, and elicitation technique routing.
- Added Lifecycle Closure, Game Studio Depth, and Test Architecture Depth with compact workflow refs, templates, replay fixtures, and validation coverage.
- Added guided closure utilities: `investigation`, `working-backwards-challenge`, `sprint-status`, `checkpoint-preview`, and `adversarial-review`.
- Expanded packaged parity replay to 58 guidance fixtures and made install smoke validate the same route matrix as the source checkout.
- Polished human-facing guide output while keeping JSON state, workflow refs, templates, evidence, and handoffs compact for future agents.
- Recorded P2 scope decisions for non-goal/deferred surfaces so future work does not pretend everything belongs in the current Codex plugin.
