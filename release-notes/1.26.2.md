# Forge Method 1.26.2

Forge Method 1.26.2 adds an anti-cache invocation contract to the skill.

## Highlights

- Every `$forge-method` invocation must run the launcher before answering.
- Agents must not answer from old chat state, stale initialization prompts, or cached assumptions.
- The current filesystem and launcher output are authoritative.

## Notes

This reduces stale same-chat behavior after updates. A chat that already loaded an older skill may still need one fresh invocation after the package updates, but future invocations should rerun the launcher instead of repeating old wording.
