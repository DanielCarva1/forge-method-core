# Forge Method 1.33.0

Forge Method Core v1.33.0 ships MDA Game Lens and manual update support:

- add `MDA Lens` to Game Studio as an integrated facilitation and quality lens, not a separate workflow
- add `mda_trace` to generated game briefs with target aesthetics, player experience hypothesis, desired dynamics, supporting mechanics, feedback/UI signals, proof/playtest, and unresolved risks
- make `artifact game-check` warn for legacy game briefs without MDA while failing incomplete MDA blocks
- route MDA, player-feeling, fun, dynamics/mechanics, and playtest prompts to the right Game Studio workflows
- update game facilitation packs, workflow state machines, and templates so later agents preserve the player-experience trace
- add `$forge-update` as an Operational Maintenance Skill for explicit Git marketplace updates and short human patch notes
- document the operational exception: `$forge-method` remains the single product entrypoint; `$forge-reload` and `$forge-update` only maintain the runtime package
