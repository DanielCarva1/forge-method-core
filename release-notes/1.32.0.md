# Forge Method 1.32.0

Guided early visual proof and platform operations coverage.

This release improves the human-guided start of user-facing products and adds a dedicated path for infrastructure, CI/CD, database, deploy, secrets, observability, and rollback work.

Highlights:

- Broad product, game, UX, and creative ideas now carry an early visual proof contract before requirements harden.
- `visual-alignment-prototype` gives the agent a concrete workflow for showing screens, design directions, or proof-of-direction examples during early discovery/spec work.
- `platform-ops` gives infra, CI/CD, database, deploy, secrets, observability, and rollback work its own guided route instead of hiding it behind generic build work.
- Discovery, spec, research, and game brief artifacts now include visual proof/reference fields so humans can correct direction earlier.
- Verification scripts now use the responsive per-test runner, with progress, timeouts, JSON/JUnit reports, debug logs, and slowest-test reporting.
