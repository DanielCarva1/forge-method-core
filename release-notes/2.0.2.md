# Forge Method Core 2.0.2

**Released:** 2026-06-23

Hotfix for the update path.

## Fixes

- `$forge-update` now resolves the installed plugin package from the personal Codex marketplace before printing version summaries.
- Version detection now prefers `VERSION` when it is newer than `.codex-plugin/plugin.json`.
- Legacy skill installs no longer refresh to the v2 package while showing stale `1.34.1` patch notes.

## Upgrade

```txt
$forge-update
```
